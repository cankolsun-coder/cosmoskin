# COSMOSKIN Mobile Design Package for Claude Design

This package is intentionally reduced below 50MB for Claude Design. It includes only the files needed for mobile UI/UX redesign:

- Core frontend HTML pages
- Product, category, brand, account, auth, legal and checkout pages
- CSS and JavaScript used by the frontend
- Essential product, hero, editorial, brand, icon and payment assets
- Reference UI mockups from the current mobile design direction

Excluded intentionally:

- .git history
- admin/backend/serverless functions
- Supabase/database files
- automation scripts
- large guide/fallback image folders
- reports and changelogs

Instruction for Claude:
Use this ZIP as the source of truth for COSMOSKIN mobile redesign. Do not redesign desktop. Redesign the mobile website experience only, using the reference mockups as the visual quality benchmark.
